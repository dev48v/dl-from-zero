↑ Diffusion learns to REVERSE a gradual noising process. You train by adding noise to real data and teaching a network to predict (and remove) it; you generate by starting from pure noise and denoising step by step. This is exactly how Stable Diffusion and DALL·E create images.

🔨 A diffusion model, in code

    
Pick a noise schedule → add noise to a real image at a random step → train a U-Net to predict that noise → minimise MSE → sample by denoising from pure noise step by step. ~7 steps. This is the real DDPM training loop; in practice you reach for the diffusers library, but every line below is what it does under the hood.

    
    
      
  1. 1
    Pick a noise schedule (betas → alphas)
    
        
    import torch, torch.nn as nn, torch.nn.functional as F

T = 1000                                  # number of diffusion steps
betas       = torch.linspace(1e-4, 0.02, T)    # how much noise per step
alphas      = 1.0 - betas
alpha_bar   = torch.cumprod(alphas, dim=0)     # cumulative product ᾱ_t
# ᾱ_t controls how much of the ORIGINAL image survives at step t
sqrt_ab     = alpha_bar.sqrt()
sqrt_1m_ab  = (1 - alpha_bar).sqrt()
    
          
    2. 

      
  2. 2
    Forward process — add noise in one shot
    
        
    # instead of looping t times, the schedule lets us jump straight to step t:
#   x_t = sqrt(ᾱ_t) * x_0  +  sqrt(1-ᾱ_t) * noise
def add_noise(x0, t, noise):
    ab  = sqrt_ab[t].view(-1, 1, 1, 1)
    omb = sqrt_1m_ab[t].view(-1, 1, 1, 1)
    return ab * x0 + omb * noise           # the noisy image x_t
    
          
    3. 

      
  3. 3
    The network: a U-Net that predicts the noise
    
        
    # the model takes the noisy image x_t and the step t,
# and predicts the noise epsilon that was added at that step.
model = UNet(in_ch=3, out_ch=3, time_emb=True)   # same in/out shape as image
# (a U-Net = encoder that downsamples + decoder that upsamples,
#  with skip connections; t is fed in as a time embedding)
eps_pred = model(x_t, t)        # predicted noise, same shape as the image
    
          
    4. 

      
  4. 4
    Training objective — just MSE on the noise
    
        
    opt = torch.optim.Adam(model.parameters(), lr=2e-4)

def train_step(x0):
    t     = torch.randint(0, T, (x0.size(0),))   # random step per image
    noise = torch.randn_like(x0)                 # the noise we add
    x_t   = add_noise(x0, t, noise)              # corrupt the image
    loss  = F.mse_loss(model(x_t, t), noise)     # predict the noise!
    opt.zero_grad(); loss.backward(); opt.step()
    return loss.item()           # no adversarial game — one stable loss

    6. 

      
  6. 6
    Sampling — denoise from pure noise, step by step
    
        
    @torch.no_grad()
def sample(shape):
    x = torch.randn(shape)                    # start from PURE noise (t = T)
    for t in reversed(range(T)):              # walk back T -> 0
        eps = model(x, torch.tensor([t]))     # predict the noise in x
        a, ab = alphas[t], alpha_bar[t]
        # remove the predicted noise to estimate the slightly cleaner x_{t-1}
        x = (x - (1-a)/(1-ab).sqrt() * eps) / a.sqrt()
        if t > 0:
            x = x + betas[t].sqrt() * torch.randn_like(x)   # add a little back
    return x                                  # a brand-new image
