🔨 The three update rules

    
Each is a small variation on "step against the gradient".

    
    
      
  1. 1
    SGD — plain gradient step
    w = w - lr * grad;
// simple, but in a narrow valley it bounces wall-to-wall
    2. 
      
  2. 2
    Momentum — a heavy ball
    v = beta * v - lr * grad;   // velocity accumulates
w = w + v;                  // keeps rolling in a consistent direction
// damps the zigzag, builds speed along the valley floor
    3. 
      
  3. 3
    Adam — adaptive, per-parameter
    m = b1*m + (1-b1)*grad;          // 1st moment (momentum)
v = b2*v + (1-b2)*grad**2;       // 2nd moment (per-dim scale)
w = w - lr * m̂ / (sqrt(v̂) + eps); // big steps where flat, small where steep
    4. 
      
  4. 4
    Which to use?
    // Adam: the default for most deep learning — fast, forgiving of lr
// SGD + momentum: often generalizes a touch better; common for vision
// the optimizer is a hyperparameter, like the learning rate
