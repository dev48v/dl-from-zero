🛣️ ResNet & Skip Connections — The +1 Gradient Highway

Yesterday the gradient died in a deep net. Add one +x and it survives.

Day 21 showed how gradients vanish exponentially in deep networks. ResNet's fix (2015) is almost absurdly simple: instead of a block computing y = F(x), make it compute y = x + F(x) — add the input straight back to the output through a skip connection. In backprop that turns each block's local factor from F'(x) into 1 + F'(x), and that +1 gives the gradient a highway straight back to the early layers. This is why 152-layer networks suddenly became trainable.

🔨 Build it yourself

    
The whole idea is one line — add the input back. Here's a residual block in PyTorch and the gradient-product intuition it fixes.

    
    
      
  1. 
        
    1
    A plain block: y = F(x)
    
        
    A normal block transforms its input completely: two linear/conv layers with an activation. Stack many and the backward pass multiplies each block's derivative — small derivatives compound to nearly zero (vanishing).
    
        
    class PlainBlock(nn.Module):
    def __init__(self, d):
        self.f = nn.Sequential(nn.Linear(d,d), nn.ReLU(), nn.Linear(d,d))
    def forward(self, x):
        return F.relu(self.f(x))       # y = F(x)

  2. 
        
    2
    A residual block: y = x + F(x)
    
        
    The only change: add the input back before the final activation. Now the block learns a residual — the small correction on top of "pass the input through unchanged" — which is far easier to learn than a full mapping, especially when the ideal is close to identity.
    
        
    class ResidualBlock(nn.Module):
    def __init__(self, d):
        self.f = nn.Sequential(nn.Linear(d,d), nn.ReLU(), nn.Linear(d,d))
    def forward(self, x):
        return F.relu(x + self.f(x))   # y = x + F(x)  ← the skip

  3. 
        
    3
    Why the +1 saves the gradient
    
        
    Differentiate y = x + F(x): dy/dx = 1 + F'(x). Backprop multiplies these across blocks. The 1 means even if F'(x) is tiny, the factor stays near 1 instead of near 0 — so the product doesn't collapse. The gradient flows through the identity path like a highway.
    
        
    // plain:    dL/dx₁ = Π f'(zₗ)          → 0   (each f' ≤ ~0.25)
// residual: dL/dx₁ = Π (1 + f'(zₗ))    → stays ~O(1)
//                          ↑ the identity path never vanishes

  4. 
        
    4
    When dimensions change: projection shortcut
    
        
    If a block changes the feature size (e.g. a conv that halves spatial size / doubles channels), x and F(x) no longer match, so you can't just add them. Use a 1×1 conv (or linear) on the skip to project x into the new shape before adding — the "projection shortcut" from the ResNet paper.
    
        
    def forward(self, x):
    identity = self.proj(x) if self.proj else x   # 1x1 conv if shapes differ
    return F.relu(identity + self.f(x))
