🚪 LSTM & GRU — Gates That Remember the Long Ago

Drop a memory early — see who still has it 20 steps later

Press Remember to write a value into the cell, then feed noise. The LSTM keeps a near-flat cell state (its forget gate stays open), holding the value. A plain RNN bleeds it away within a few steps.

↑ Both start blank. Hit "Remember", then feed noise. The LSTM's gate decides to KEEP (≈multiply by 1), so its memory rides the conveyor belt unchanged. The RNN multiplies its state by <1 every step → it fades. That's the vanishing-gradient cure.

    Forget gate: what to drop
    const f = sigmoid(Wf·[h, x]);   // 0 = erase, 1 = keep
