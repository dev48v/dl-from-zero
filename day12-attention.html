🔭 Attention — How Transformers Read Everything at Once

Click a word — watch it decide who to pay attention to

Pick a query word below. The demo computes a real dot-product score from that word to every word, scales by √d, runs softmax, and the weights (which sum to 1) become the bars. Notice "it" attends strongly back to "cat".

↑ Attention lets each word look at every other word and decide how much to "pay attention" to it. The weighted sum of all words' values becomes that word's new context vector — the core operation inside every Transformer.

🔨 Self-attention, in code

    
Embed tokens → project to Q, K, V → score with QKᵀ/√d → softmax → weighted sum of V. ~7 steps. This exact block is the heart of GPT and BERT.

    
    
      
  1. 1
    Embed the tokens
    
        
    // each word id → a vector of length d_model (learned)
const X = tokens.map(id => embedding[id]);   // shape: [seqLen, d_model]
const d = X[0].length;                       // model dimension
    
          
    2. 

      
  2. 2
    Project to Query, Key, Value
    
        
    // three learned weight matrices give each token a Q, K and V
const Q = matmul(X, Wq);   // "what am I looking for?"
const K = matmul(X, Wk);   // "what do I offer?"
const V = matmul(X, Wv);   // "what do I pass on if chosen?"
    
          
    3. 

      
  3. 3
    Score = QKᵀ / √d
    
        
    // dot product of every query with every key = similarity grid
let scores = matmul(Q, transpose(K));        // [seqLen, seqLen]
scores = scores.map(row => row.map(s => s / Math.sqrt(d)));  // scale
    
          
    4. 

      
  4. 4
    Softmax each row → attention weights
    
        
    function softmax(row){
  const m = Math.max(...row);                 // for numerical stability
  const e = row.map(s => Math.exp(s - m));
  const sum = e.reduce((a,b)=>a+b, 0);
  return e.map(x => x / sum);                 // weights, each row sums to 1
}
const weights = scores.map(softmax);          // [seqLen, seqLen]
    
          
    5. 

      
  5. 5
    Weighted sum of V → context vectors
    
        
    // each token's output = blend of ALL values, mixed by its weights
const out = weights.map(w =>
  V[0].map((_, c) => w.reduce((acc, wi, j) => acc + wi * V[j][c], 0))
);   // out[i] is token i's new, context-aware representation
    
          
    6. 

      
  6. 6
    Multi-head: run it several times in parallel
    
        
    // split d into h heads; each head learns its own Wq/Wk/Wv "view"
const heads = [];
for (let i = 0; i < numHeads; i++)
  heads.push(attention(X, Wq[i], Wk[i], Wv[i]));   // syntax → meaning → ...
const multi = matmul(concat(heads), Wo);            // combine the views
    
          
    7. 

      
  7. 7
    Add positional encoding (order matters)
    
        
    // attention alone is order-blind — inject position before it
X = X.map((vec, pos) => vec.map((v, i) =>
  v + (i % 2 === 0 ? Math.sin(pos / 10000 ** (i/d))
                   : Math.cos(pos / 10000 ** ((i-1)/d)))
));   // now "dog bites man" ≠ "man bites dog"
