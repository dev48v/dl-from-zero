📉 Vanishing & Exploding Gradients — Why Deep Nets Stall

Watch the gradient multiply its way to nothing — or blow up — through a deep chain

Backprop hands the gradient backwards through the layers, and at every layer it gets multiplied by that layer's local factor |w| · f'(x). This demo builds a real chain of N layers, does a genuine forward pass to get each layer's pre-activation, then computes the real chain-rule product of those local factors from the output back to the input. Each bar is the gradient magnitude that survives at that layer. Pick a deep chain with Sigmoid — whose derivative never exceeds 0.25 — and watch the bars shrink to essentially zero at layer 1 (vanishing). Crank the weight scale up and the product blows past 1 every step (exploding). Switch to ReLU + He init and the chain stays close to 1 the whole way down — the fix. Start with Sigmoid and depth 16: each layer's factor is at most 0.25, so the running product collapses — the bar at layer 1 is essentially 0. That's vanishing: the first layers get almost no learning signal. Now drag weight scale to ~2.5 — the factor jumps above 1 and the product explodes. Finally hit He init and pick ReLU: the factors sit near 1 and the gradient survives all the way down.

↑ Backprop is a long product of per-layer terms. If each term is a little below 1, the product shrinks exponentially with depth (vanishing) and the early layers barely move; if each is above 1, it grows exponentially (exploding) and training diverges. Squashing activations like sigmoid (derivative ≤ 0.25) and tanh (≤ 1) practically guarantee shrinkage in deep stacks — which is exactly why deep nets stalled for years. ReLU (derivative 0 or 1) plus He initialization keeps the per-layer factor near 1, and batch norm (Day 19) and residual/skip connections give the gradient a clean path so it neither dies nor blows up.

🔨 Watch a gradient vanish (and fix it), in code

    
Build a deep chain by hand → run a forward pass to get each layer's pre-activation → walk backwards multiplying by the local factor |w|·f'(x) → print the running gradient and watch it collapse with sigmoid → swap in ReLU + He init and watch it survive → add gradient clipping for the exploding case → and see how batch norm and residual connections keep the path open. ~8 short steps of plain Python — no framework.

    
    
      
  1. 1
    Build a deep chain of layers
    
        
    import numpy as np
np.random.seed(0)

N = 20                       # depth: 20 layers deep
width = 64                   # neurons per layer

# one weight matrix per layer. "scale" controls the init variance.
def make_net(scale):
    return [np.random.randn(width, width) * scale for _ in range(N)]

net = make_net(scale=1.0)    # naive: std = 1 (too big, as we'll see)
x0  = np.random.randn(width) # one input vector

  2. 2
    Forward pass — remember each pre-activation
    
        
    def sigmoid(x): return 1 / (1 + np.exp(-x))

# forward: z = W @ a  (pre-activation), then a = f(z).
# stash every z, because backprop needs f'(z) at each layer.
def forward(net, x, f):
    a, zs = x, []
    for W in net:
        z = W @ a
        zs.append(z)          # save the pre-activation
        a = f(z)
    return a, zs

out, zs = forward(net, x0, sigmoid)
    
          
    3. 

      
  3. 3
    The local factor: |w|·f'(z) at each layer
    
        
    # derivative caps: sigmoid' <= 0.25, tanh' <= 1, relu' is 0 or 1.
def sigmoid_deriv(z): s = sigmoid(z); return s * (1 - s)   # peaks at 0.25

# per-layer local gradient factor (scalar summary):
#   how much the gradient is scaled passing through this layer.
def local_factor(W, z):
    return np.abs(W).mean() * sigmoid_deriv(z).mean()
    
          
    4. 

      
  4. 4
    Backprop = the running product of those factors
    
        
    # start the gradient at 1.0 at the OUTPUT, walk backwards,
# multiply by each layer's local factor. THIS is the chain rule.
def chain(net, zs, dfn):
    g = 1.0
    mags = [g]
    for W, z in zip(reversed(net), reversed(zs)):
        g *= np.abs(W).mean() * dfn(z).mean()
        mags.append(g)
    return mags[::-1]           # mags[0] = gradient reaching layer 1

mags = chain(net, zs, sigmoid_deriv)
print("grad reaching layer 1:", mags[0])   # -> ~1e-12 : VANISHED

  5. 5
    Why it dies: a product of numbers below 1
    
        
    # sigmoid' never exceeds 0.25. Even with |w| ~ 1, each factor <= 0.25,
# so over 20 layers the product is about 0.25**19:
print(0.25**19)      # -> 3.6e-12  (essentially zero)

# a factor of 0.9 seems harmless, but 0.9**50 = 0.005 -> still tiny.
# >1 is the opposite disaster: 1.5**20 = 3325  -> the gradient EXPLODES.
print(0.9**50, 1.5**20)

  6. 6
    Fix 1 — ReLU + He initialization
    
        
    def relu(x):        return np.maximum(0, x)
def relu_deriv(z):  return (z > 0).astype(float)   # 0 or 1 (no 0.25 cap)

# He init sets the weight std so the signal variance is PRESERVED
# through a ReLU layer:  std = sqrt(2 / fan_in).
net = [np.random.randn(width, width) * np.sqrt(2/width) for _ in range(N)]
out, zs = forward(net, x0, relu)
mags = chain(net, zs, relu_deriv)
print("grad reaching layer 1:", mags[0])   # -> stays ~order 1 : SURVIVES

  7. 7
    Fix 2 — gradient clipping for the exploding case
    
        
    # exploding gradients (common in RNNs) get capped by rescaling
# the whole gradient vector if its norm exceeds a threshold.
def clip_grad(g, max_norm=5.0):
    norm = np.linalg.norm(g)
    if norm > max_norm:
        g = g * (max_norm / norm)      # same direction, safe length
    return g
# this keeps a giant step from blowing the weights to NaN.

  8. 8
    Fix 3 — batch norm & residual connections
    
        
    # Batch norm (Day 19) re-centers/rescales each layer's pre-activation,
# keeping z in the sensitive region where derivatives aren't tiny.

# Residual / skip connections (ResNet) add the input back:
def residual_block(x, layer):
    return x + layer(x)        # y = x + F(x)
# now backprop has a "+1" straight-through path: dy/dx = 1 + dF/dx,
# so the gradient can flow around the layer instead of dying in it.
# These are why we can train nets 100s of layers deep today.

Backprop multiplies a local factor |w|·f'(x) at every layer, so the gradient reaching the earliest layers is a long product. Factors below 1 make it vanish exponentially with depth; factors above 1 make it explode. Squashing activations (sigmoid ≤ 0.25, tanh ≤ 1) all but guarantee vanishing in deep stacks — which stalled the field for years. ReLU (derivative 0 or 1) with He init keeps the factor near 1; gradient clipping tames explosions; and batch norm (Day 19) plus residual/skip connections keep the path open so nets hundreds of layers deep actually train. Gates in LSTMs (Day 11) solve the same problem for sequences.
