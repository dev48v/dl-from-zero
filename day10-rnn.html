🔁 RNNs — a Network With a Memory of the Past

Same cell, fed one token at a time — the state remembers

An RNN reads a sequence step by step. Each step mixes the new input with the hidden state carried from the last step. So the output depends on the whole history, not just the current token. Build a sequence and step through it.

Reading: —

↑ The line is the hidden state over time. Notice: feed the SAME token after different history and the state lands somewhere different — that's memory. "not" then "good" ends up different from "good" then "not".
