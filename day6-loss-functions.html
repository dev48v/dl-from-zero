Each is a one-liner. The art is matching the loss to the task.

    MSE — for regression (predicting numbers)
    const mse = (pred, y) => (pred - y) ** 2;

    const mae = (pred, y) => Math.abs(pred - y);

    const bce = (p, y) => -(y*Math.log(p) + (1-y)*Math.log(1-p));
